Design PrinciplesWhat they doHow I used them
Columns Columns are implimented on most of the internets websites. They allow for a seperation of information that lets the user browse the site much easier. I used bootstrap on this website because with bootstrap, making columns very easy. Since I did not have a lot of information I opted for two colums, on smaller for links, and the other for the main body.
Navbars Nav bars allow users to traverse your website and at the same time show on what page the user is currently on. I decided to use Bootstrap so that my navbar was responsive and looked good. On smaller screens, it will collapse into a hamburger button that you can press and bring up the navbar links.
Color Scheme Color schemes provide a nice set of colors that work well together. This can make the page pleasing to look at, and easy to navigate. I chose a few muted colors that I thought went together well. The website is bright, without hurting your eyes, and the colors work well together.
